CSS Selectors & Specificity

  • element {} - Selects all instances of a tag
  • .class {} - Selects elements with a specific class
  • #id {} - Selects elements with a specific ID
  • Inline styles > ID selectors > Class selectors > Element selectors

Common CSS Properties

color: red; - Sets text color
background-color: lightgray; - Sets background color
margin: 10px; - Defines external spacing
padding: 10px; - Defines internal spacing
border: 2px solid black; - Defines border properties
