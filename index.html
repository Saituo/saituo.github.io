Advanced CSS Selectors & Specificity

  • element - Targets all instances of an element (e.g., p).
  • .class - Targets elements with a specific class (e.g., .container).
  • #id - Targets a unique element with an ID (e.g., #header).
  • attribute selectors - Example: input[type="email"].
  • pseudo-classes and pseudo-elements such as :hover, :first-child, ::before, and ::after.
