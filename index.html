Advanced CSS Selectors & Specificity

  • element - Targets all `<p>` elements, for example.
  • .class - Targets elements with a specific class, like .container.
  • #id - Targets the unique element with that ID, e.g., #header.
  • attribute selectors - For example, input[type="email"] targets email inputs.
  • pseudo-classes and pseudo-elements - e.g., :hover, :first-child, ::before, and ::after.

Different Types of Buttons

Buttons can be created in three primary ways in HTML:

  1. Standard Button (<button>):
    This element is versatileâ€”you can use it as a submit button when inside a form or as a clickable element for JavaScript actions. 
    <button type="button">Click Me</button>
  2. Submit Button (input type="submit"):
    This button submits a form when clicked. 
    <input type="submit" value="Submit Form">
  3. Reset Button (input type="reset"):
    This button resets all form fields to their initial values. 
    <input type="reset" value="Reset Form">
