Box-Shadow Examples

CSS Property Example Description
box-shadow: 4px 4px 8px gray;
Simple Shadow
 Creates a standard shadow effect.
box-shadow: inset 5px 5px 10px black;
Inset Shadow
 Creates an inner shadow effect inside an element.
box-shadow: 0 4px 15px rgba(0,0,0,0.5);
Blurred Shadow
 Applies a soft, blurred shadow with transparency.

Pseudo-Classes and Pseudo-Elements

  • :hover - Changes styles when an element is hovered.
  • :focus - Applies styles when an element (like an input) is focused.
  • ::before - Inserts content before an element.
  • ::after - Inserts content after an element.
  • ::first-letter - Styles the first letter of text.

Button Hover Effects

CSS Property Example Description
button:hover { background-color: red; } Changes button color on hover.
a:hover { text-decoration: underline; } Hover Over This Link Adds an underline effect when hovering over links.
